我的第一页

发表于 2016-12-01

非程序员，对Github的版本控制需求不大，但用Github pages做博客记录，还是不错的。

记录一下本博客在Github上安装的过程。

安装全部在WINdows下进行，基本有以下步骤

  1. 注册Github；
  2. 创建 用户名.Github.io 的库；
  3. 安装Node.js；
  4. 安装 Git for Windows；
  5. 安装Hexo；
  6. 更换Hexo主题

注册Github

注册后，需通过注册邮箱验证；
注册免费的公共库即可

创建库

创建 用户名.Github.io的库，这样才可以 https://用户名.Github.io的方式访问站点；

安装Node.js

从这里下载相应版本；

默认安装即可

安装完成后可通过CMD进行验证

验证命令：

node -v

npm -v

安装 Git for Windows

从这里 下载安装相应版本

安装时，需加入系统环境变量

安装Hexo

在本地电脑硬盘建立工作目录，如D:/hexo，打开该文件夹，右键菜单选择“Git Bach Here”

在打开的命令窗口运行以下命令

npm install hexo-cli -g

npm install hexo --save

hexo -v // 验证版本

hexo init // 初始化

npm install // 安装

运行Hexo

在Git窗口运行

hexo g

hexo s

在浏览器地址栏输入　http://localhost:4000

# hexo
